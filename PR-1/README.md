# TechFlow Website

TechFlow is a modern multi-page website built using **Node.js** and **HTML/CSS**.  
The project uses a custom Node.js server to serve different pages with clean routing and a modern UI.

---

## Features

- Node.js HTTP server
- Multi-page navigation
- Home, About, Services, Contact pages
- Custom 404 error page
- Modern dark UI design
- Responsive layout

---

## Technologies Used

- Node.js
- HTML5
- CSS3
- JavaScript

---

## Project Structure

```
project/

├── index.html

├── 404.html
│
└── README.md
```

---

## How to Run

1. Make sure Node.js is installed
2. Open terminal in the project folder
3. Run:

```bash
npm start
```

4. Open browser and go to:

```
http://localhost:8000
```

---

## Routes

| URL | Page |
|----|------|
| / | Home |
| /about | About |
| /services | Services |
| /contact | Contact |
| Others | 404 Page |
